# Screenshots

This folder contains screenshots for use in `README.md` and other documentation.

## Adding screenshots

1. Take screenshots at 1280 × 800 or higher resolution.
2. Save as PNG.
3. Use descriptive, lowercase, hyphenated filenames (e.g. `catalogue-browse.png`).
4. Add a row to the table in `README.md` under the **Screenshots** section.
