Backup

Back up your catalogue database and embroidery design files to folders of your choice. The database backup saves your catalogue data, settings, tags, and projects. The designs backup saves the actual embroidery files.

Important

The database and embroidery designs are backed up separately. To make a complete backup of your catalogue, make sure you run both backups.

Backup Destinations

Set separate destination folders for the database and designs backups.

Choose where the database backup file should be saved. This backup contains your catalogue data only — not the embroidery design files. Timestamped .db files (e.g. catalogue_2026-04-06_1430.db) will be written here.

Choose where the embroidery design files should be backed up. This backup contains the files only — not the catalogue database. New and changed files are mirrored here incrementally. Files removed from the source are moved to a dated _deleted\YYYY-MM-DD\ archive folder inside the destination, not permanently deleted.
